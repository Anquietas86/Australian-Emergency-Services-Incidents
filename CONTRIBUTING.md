# Contributing to Australian Emergency Services Incidents

Thanks for your interest in contributing!

## How to contribute

1. **Fork** the repository on GitHub.
2. Create a **feature branch** for your work.
3. Make your changes following the [Home Assistant developer documentation](https://developers.home-assistant.io/).
4. Submit a **Pull Request** with a clear description of your changes.

## Code style
- Follow Home Assistant core code style (Black, isort).
- Use meaningful commit messages.

## Issues
- Search existing issues before creating a new one.
- Include relevant logs, Home Assistant version, and integration version.

## Adding new states
When adding support for a new state:
- Use the official hazard/incident API for that state.
- Keep the data processing consistent with existing state modules.
- Update README.md, info.md, and release notes.
